# 🐱 猫捉老鼠游戏 🐭

一个简单有趣的 Python 小游戏，用鼠标控制猫去捉老鼠！

## 🎮 游戏规则

- 🐱 **猫**：跟随你的鼠标移动
- 🐭 **老鼠**：在屏幕上随机移动
- 🎯 **目标**：用猫捉住所有的老鼠
- 💯 **计分**：每捉住一只老鼠获得 10 分

## ⌨️ 操作方式

| 操作 | 说明 |
|------|------|
| 🖱️ 鼠标移动 | 控制猫的位置 |
| R 键 | 重新开始游戏 |
| ESC 键 | 退出游戏 |

## 🚀 安装和运行

### 1. 克隆仓库
```bash
git clone https://github.com/zccheng77/cat-mouse-game.git
cd cat-mouse-game
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 运行游戏
```bash
python game.py
```

## ✨ 游戏特性

✅ **简单易上手** - 直观的游戏机制
✅ **实时反馈** - 显示分数和老鼠数量
✅ **智能 AI** - 老鼠会随机移动和改变方向
✅ **递增难度** - 每次捉住所有老鼠后增加难度
✅ **流畅体验** - 60 FPS 游戏帧率
✅ **友好界面** - 清晰的视觉反馈

## 🎨 游戏画面说明

- 🟠 **橙色圆圈**：猫（跟随鼠标）
- 🔴 **红色圆圈**：老鼠（随机移动）
- **Score**：已捉住的老鼠数 × 10
- **Mice**：当前屏幕上的老鼠数

## 🎲 难度系统

- **初始难度**：屏幕上有 3 只老鼠
- **难度提升**：每次捉住所有老鼠后，会增加更多老鼠
  - 第一关胜利：4 只老鼠
  - 第二关胜利：5 只老鼠
  - 以此类推...

## 💻 技术栈

- **语言**：Python 3.7+
- **库**：Pygame 2.5.2
